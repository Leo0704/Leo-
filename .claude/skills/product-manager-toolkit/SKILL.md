# Skill: 产品经理工具包

当用户要求进行产品管理相关工作时使用。

## 使用场景

- "创建 PRD"
- "分析用户反馈"
- "优先级排序"
- "规划路线图"

## 功能

### 1. 需求文档 (PRD)
帮助创建产品需求文档：
- 功能描述
- 用户故事
- 验收标准
- 技术考虑

### 2. 客户反馈分析
分析用户反馈：
- 提取痛点
- 识别需求
- 优先级建议

### 3. RICE 优先级
使用 RICE 方法评估：
- Reach (影响范围)
- Impact (影响程度)
- Confidence (信心程度)
- Effort (工作量)

### 4. 用户研究
协助用户研究：
- 访谈问题
- 调查设计
- 数据分析框架

## 输出格式

根据任务类型输出相应格式的文档。
