# Skill: 查找技能

帮助用户发现和使用 Claude Code 技能。

## 使用场景

- "有什么技能可以用"
- "怎么实现 X 功能"
- "有没有做 Y 的技能"

## 功能

### 1. 技能发现
列出所有可用的 skills：
- 从 `~/.agents/skills/` 加载
- 搜索技能描述

### 2. 技能推荐
根据用户需求推荐合适的技能：
- 分析需求
- 匹配技能
- 提供使用建议

### 3. 技能安装
帮助安装新技能：
- 解释技能格式
- 提供模板
- 指导安装步骤

## 使用方式

当用户询问可用技能时：
1. 加载所有 skills
2. 展示可用技能列表
3. 解释每个技能的用途

当用户需要特定功能时：
1. 搜索匹配的技能
2. 提供技能信息和用法
