# 项目理想状态

## 产品愿景
一个让 Claude Code 能够**自主地**把软件开发到理想状态的工作流框架。

## 核心价值
- **目标驱动** - 不告诉 AI 做什么，而是告诉 AI 你想要什么
- **保留独特价值** - 验收标准自动验证
- **依赖原生能力** - 用原生 Task 系统，更强大
