# 待办事项应用示例

这是一个使用**目标驱动工作流**开发的示例项目。

## 如何使用

1. 在浏览器中打开 `index.html`
2. 输入待办事项，按回车或点击"添加"
3. 点击复选框标记完成
4. 点击"删除"移除事项

## 工作流演示

### 理想状态 (.workflow/GOAL.md)
描述了我们想要构建什么：
- 添加/完成/删除功能
- 本地持久化
- 简洁美观

### 当前状态 (.workflow/REALITY.md)
记录了实际实现的内容和进度。

## 特点

- 纯前端，无需服务器
- localStorage 持久化
- 响应式设计
- 代码 < 150 行

## 自己试试

1. 复制 `.workflow/` 目录到你的项目
2. 编辑 `GOAL.md` 描述你的理想状态
3. 在 Claude Code 中说"继续开发"
4. 观察项目逐步接近理想状态
