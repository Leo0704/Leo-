# 继续开发提示

当你想要让 Claude Code 继续开发时，使用以下提示：

---

## 基础提示

```
请阅读 .workflow/STATUS.md，了解当前项目状态，然后继续完成下一个任务。
```

---

## 详细提示

```
请按以下步骤继续开发：

1. 读取 .workflow/STATUS.md 了解当前进度
2. 读取 .workflow/tasks.json 查看任务列表
3. 选择下一个待处理的任务
4. 执行该任务
5. 完成后更新状态文件
6. 如果会话还有空间，继续下一个任务

工作准则：
- 每完成一个任务就 commit
- 更新 STATUS.md 和 status.json
- 遇到问题记录到 decisions.md
```

---

## 新会话提示

每次开始新会话时：

```
继续开发。请读取 .workflow/ 状态文件并继续工作。
```

---

## 检查进度提示

```
显示当前项目进度和下一个任务
```

---

## 添加新任务提示

```
在 .workflow/tasks.json 中添加以下任务：
- 任务标题: [标题]
- 描述: [描述]
- 优先级: [数字]
```
