## 技术栈

- **前端**: React + Vite + Tailwind CSS
- **后端**: Node.js + Express
- **数据库**: SQLite

## 快速开始

### 安装依赖

```bash
./init.sh
```

或手动安装：

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

应用将在 http://localhost:3000 启动

### 构建

```bash
npm run build
```
